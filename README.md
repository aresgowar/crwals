# Crawl Data App Restful API 

Simple Restful API that is crawl data from https://stackoverflow.com/questions/tagged/ibm-blockchain using Golang and PostgreSQL.

## API Routes

### GET /

Get all records

### GET /:id

Get records by ID

### POST /

Add new record

### PUT /:id

Edit record by ID

### DELETE /:id

Delete record by ID
